## **EXPENSE MANAGER V1.0**

A browser-based Expense Manager built using HTML, CSS, and Vanilla Java Script that helps users track their expenses, manage budgets and visualize spending through charts and a calendar view. Data is stored locally using localStorage, so no backend is required.



Features:

* Dashboard Overview:
*  	Total Balance, Income and Expenses
*  	Recent Transaction history with expand/collapse
*  	Delete Transaction directly from history
* Calendar View:
*  	Monthly Expense visualization
*  	Color-coded spending levels(Low/Moderate/High)
*  	Click any day to view detailed expenses
* Smart Budget Alerts:
*  	Set monthly budgets per category
*  	Alerts show whether spending is: On track, Near Limit, Exceeded
*  	Calculated dynamically for the current month
* Settings Panel:
*  	Add/delete custom categories
*  	Set category-wise budgets
*  	Change Currency
*  	Download data as CSV
*  	Reset all stored data
*  	Light/Dark mode toggle
* UI/UX:
*  	Responsive Layout
*  	Clean Settings UI
*  	Smooth Interactions
*  	Dark/Light theme support



How to Run Locally?

1. Clone the repository

 	git clone https://github.com/RupinderSingh2007/Expense-Manager.git

2. Open the project folder

3. Open *index.html* in your browser

OR

just search on google:
  expensemanager-rs.netlify.app



Data Persistence:

* All data is stored in the browser using localStorage
* No data is sent to any server
* **Clearing browser storage will reset the app**
